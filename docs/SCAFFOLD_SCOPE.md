# Full Scaffold Scope

## Purpose
This branch will contain the complete roadmap-first file and folder structure for the Experimenters runtime. Every scaffold file contains a purpose-specific development roadmap and remains non-operational until its tasks, tests, and review gates are completed.

## Repositories
- QSO-GENOMES: contracts, roles, genomes, workflows, policies, manifests, migrations, and validators.
- QSO-FABRIC: orchestration, messaging, provenance, freeze controls, governance, reporting, and CLI.
- QuantumStateObjects: reusable bounded object model and lifecycle primitives.
- QSO-STUDIO: human review, visualization, inspection, and approval interface.

## Release rule
Scaffold presence is not implementation completion. No capability, compatibility, deployment, or safety claim is valid until exact-head tests and human review are recorded.
